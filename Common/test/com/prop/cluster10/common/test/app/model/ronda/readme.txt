Aparece un menú con diferentes opciones. De la 1 a la 5 simplemente crean instancias de objetos o recojen instancias de objetos i no requieren ninguna acción por parte del usuario. 

Op6: Se introduce el numero de opción, en este caso 6, acto seguido el numero de jugadores que se quiere insertar. Luego para cada jugador se debe insertar en orden el nombre y el cash inicial. Ejemplo 6 2 juan 4000 pere 2500.

Op8: Se introduce el nombre del jugador que se quiere eliminar de la lista de jugadores. Ejemplo: 8 juan

El resto de operaciones son llamadoras.
